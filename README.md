# 🕵️ helse-spane 🕵️

Spane forsøker gjøre visning og søk i dokumentasjon av juridiske vurderinger, som er gjort automatisk med referanse til paragraf og med relevante input- og output-data, for å kunne spore opp gitte vurderinger gjort på et vedtak, eller vedtak berørt av en vurdering.

## Komme i gang
